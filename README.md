# Rust Builder 镜像

## Features

1. 基于`rust`官方构建镜像；
1. 仓库使用中国科技大学的镜像[crates.io](http://mirrors.ustc.edu.cn/help/crates.io-index.html)；
1. 仓库使用腾讯云镜像[debian10.x（buster）](https://mirrors.tencent.com/help/debian.html)；
1. 安装musl-dev,musl-tools依赖；
1. 安装libpq-dev,libsqlite-dev依赖；
1. 安装libssl-dev依赖；
1. 安装linux-libc-dev依赖；
